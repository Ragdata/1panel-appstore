## [Overview](#top) üö©

VERT is a file conversion utility that uses WebAssembly to convert files on your home server instead of the cloud.  This version includes the video conversion daemon as well, to make this a complete file conversion solution.

[`^ Top`](#top)

## [Features](#top) ‚ú®

- Convert files directly on your own server using WebAssembly
- No file or size limits
- Convert images, audio, documents, and video
- Supports over 250+ file formats
- Conversion settings
- User-friendly interface built with Svelte
