## WUD (What's Up Docker)

WUD (What's Up Docker) is an open-source Docker container update monitoring tool that can monitor whether container images have been updated, and will either notify users or automatically update.

## [Usage](#top) ⭐

### Check the [documentation](https://getwud.github.io/wud/#//) for more information.

#### Default Username / Password

Username: admin
Password: Wub

To change your password, please modify the parameters during configuration.

**NOTE:** The password is encrypted using the htpasswd algorithm and can be generated with the following command:

```bash
htpasswd -nib <username> <password>
```

OR you can use the online tool to generate it at https://htpasswd.utils.com/
